# Contributing to Vinted Turbo Scraper

Thanks for your interest in contributing! This project is open to bug reports, feature requests, and documentation improvements.

## Bug Reports

1. Check [existing issues](https://github.com/Boo-n/vinted-turbo-scraper/issues) first
2. Include:
   - Search URL(s) used (anonymized if needed)
   - Expected vs actual output
   - Error messages or logs
   - Apify run ID (if on cloud)

## Feature Requests

Open an issue with `[Feature Request]` in the title. Describe:
- The use case (who needs it and why)
- Example input/output if applicable
- Any Vinted domain constraints

## Pull Requests

This repo contains documentation only. The actor source code lives on the Apify platform.
PRs accepted for:
- README improvements
- New integration examples in `/examples`
- Documentation corrections
- New FAQ entries

## Code of Conduct

- Be respectful and constructive
- No scraping of private data or PII
- Respect Vinted's Terms of Service
