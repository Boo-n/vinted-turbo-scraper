# ⚡ Vinted Turbo Scraper

Fast Vinted search URL to structured data exporter.

**Actor Store:** [apify.com/kazkn/vinted-turbo-scraper](https://apify.com/kazkn/vinted-turbo-scraper)

## What It Does

Paste a filtered Vinted search URL → Run → Export JSON/CSV/Excel in seconds.

- **~500 items/min** extraction speed
- **Batch multiple URLs** in a single run
- **26 Vinted country domains** supported
- **$0.0015/result** ($1.50 per 1,000 listings)
- **$5 free monthly credits** on Apify

## Quick Start

1. Build your search on [vinted.com](https://www.vinted.com) with filters (brand, size, price, etc.)
2. Copy the URL from your browser
3. Paste it into the [Actor](https://apify.com/kazkn/vinted-turbo-scraper) and click **Run**
4. Download CSV, JSON, or push to Google Sheets
