# Vinted Turbo Scraper — Integration Examples

This folder contains ready-to-use code snippets for integrating **Vinted Turbo Scraper** into your own projects.

| File | Language | Description |
|------|----------|-------------|
| `python_single_url.py` | Python | Call the actor with one search URL and export to CSV |
| `python_batch_urls.py` | Python | Batch multiple brands/countries in a single run |
| `nodejs_single_url.js` | Node.js | Simple API call with pretty-printed results |
| `curl_quickstart.sh` | cURL | Quick one-liner to trigger a run from terminal |
| `batch_input_example.json` | JSON | Reference input schema for batch URL runs |

## Prerequisites

- Apify API token (get it at [console.apify.com](https://console.apify.com/account#/integrations))
- For Python: `pip install apify-client`
- For Node.js: `npm install apify-client`
